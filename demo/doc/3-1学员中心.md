# 我的订单
本页面显示学员的订单信息
分为"头部" 和 订单列表两个部分

订单的具体信息部分参考.
http://coding.imooc.com/pay/myorder

+ 全部订单 | 待支付 | 报名成功 | 支付超时 | 已撤单 |
+ 订单列表
    + 订单编号 | 订单时间
    + 用户名
    + 课程名称
    + 课程价格
    + 课程状态
    + 操作按钮(不同状态的订单有不同的操作按钮)
